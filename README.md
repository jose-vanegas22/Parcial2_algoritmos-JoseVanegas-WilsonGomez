# Parcial2_algoritmos-JoseVanegas-WilsonGomez

## Descripción

Solución de 2 ejercicios de HackerRank para el examen práctico de Algoritmos. Los ejercicios fueron tomados y ejecutados directamente en la plataforma HackerRank.

## Ejercicios

### 1. Binary Search Tree : Insertion (Trees / Recursion-Iteration)

Dado un puntero a la raíz de un árbol binario de búsqueda, insertar un valor en su posición correcta y retornar la raíz actualizada.

**Complejidad:** `O(n)` — donde n es la altura del árbol; en el peor caso `O(n)`.

### 2. Jesse and Cookies (Heap / Priority Queue)

Dado un arreglo de galletas con distinta dulzura, combinar repetidamente las dos menos dulces hasta que todas alcancen una dulzura mínima k, y contar el número de operaciones necesarias.

**Complejidad:** `O(n log n)` — dominada por las operaciones `heappop` y `heappush` sobre el heap, repetidas hasta n veces.

## Cómo ejecutar

Los archivos fueron desarrollados y probados en **HackerRank**.

2. Buscar el ejercicio correspondiente
3. Copiar el código del archivo `.py` en el editor
